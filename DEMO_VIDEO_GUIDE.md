# RemitAI Demo Video Guide

## Overview
This guide helps you create a compelling demo video showcasing RemitAI's end-to-end remittance workflow on Cardano.

## Demo Flow (5-7 minutes)

### Scene 1: Introduction (0:00 - 0:15)
- Show RemitAI logo and brand
- Brief description: "AI-powered international remittances on Cardano"
- Tagline: "Fast, Cheap, Trustless"

### Scene 2: Dashboard Overview (0:15 - 0:45)
- Navigate through the main dashboard
- Highlight key metrics: Exchange rates, fees, processing speed
- Show the mobile-first responsive design
- Display wallet connection status

### Scene 3: Wallet Connection (0:45 - 1:30)
- Click "Connect Wallet"
- Show wallet selection dialog with Nami, Eternl, Flint options
- Simulate wallet connection (use mock wallet for demo)
- Display connected wallet with address and balance
- Show balance in ADA and USD equivalent

### Scene 4: Chat Interface (1:30 - 2:15)
- Open chat tab
- Ask AI questions: "What's the best rate to Philippines?"
- Show AI responses with real-time information
- Ask about fees and processing time
- Demonstrate conversational UX

### Scene 5: DEX Rate Aggregation (2:15 - 3:00)
- Navigate to Rates tab
- Show live rates from Minswap and SundaeSwap
- Highlight: volumes, liquidity, exchange rates
- Show trending indicators (green up, red down)
- Compare rates between different DEXes

### Scene 6: Creating a Remittance (3:00 - 4:30)
- Go to "Send" tab
- Enter amount: 100 ADA
- Select destination country: Philippines
- Click "Get Quote"
- Show live quote with exchange rate, fees, receive amount
- Enter recipient phone number
- Review transaction details
- Click "Send via Masumi"
- Show processing animation

## Recording Tips

### Technical Setup
- Use screen recording software (OBS, Loom, or Screenflow)
- Record at 1080p resolution (16:9 aspect ratio)
- Use a clean, clutter-free desktop
- Ensure good lighting and audio quality

### Visual Effects
- Use smooth transitions between sections
- Add background music (royalty-free, non-intrusive)
- Include text overlays for key features
- Use cursor highlighting for important UI elements
- Add captions/subtitles for accessibility

### Narration
- Speak clearly and at moderate pace
- Use a professional microphone
- Explain what you're doing before doing it
- Keep technical jargon to a minimum
- Be enthusiastic about the product

### Best Practices
- Keep total runtime between 5-7 minutes
- Start and end with strong hooks
- Show real data, not just dummy values
- Zoom into important UI elements
- Use consistent branding and colors
- Include timestamps in video description

## Editing Checklist

- [ ] All sections recorded
